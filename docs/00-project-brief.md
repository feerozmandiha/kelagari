# خلاصه پروژه کلاگری (Kelageri)

## معرفی کلی
**کلاگری** یک پلتفرم تخصصی برای طراحی، اشتراک‌گذاری و چاپ سه‌بعدی است که با هدف ایجاد یک اکوسیستم کامل برای طراحان، تولیدکنندگان و مشتریان نهایی طراحی می‌شود.

## شعار
> از خاک تا اثیر، از ایده تا شیء

## اهداف اصلی
- کتابخانه غنی مدل‌های سه‌بعدی
- نمایشگر سه‌بعدی تحت وب با Three.js
- بازار آنلاین خرید و فروش مدل‌ها
- انجمن تخصصی طراحان
- دو‌زبانه (فارسی/انگلیسی) با اولویت فارسی

## مخاطبان پروژه
- طراحان سه‌بعدی
- مهندسان مکانیک
- طراحان صنعتی
- صنعت‌گران
- دانشجویان
- کاربران عادی
